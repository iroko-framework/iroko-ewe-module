# Iroko Framework: Ewé Module

This repository publishes the Iroko Framework vocabulary for Ewé (ethnobotany) in resolvable web form.

## Namespace (draft)
Terms are published under:
- [`/vocab/`](vocab/) — machine-readable RDF files
- [`/terms/`](terms/) — human-readable term documentation

## Machine-readable vocabulary
- [Turtle (RDF)](vocab/iroko-ewe.ttl)
  
## Term pages
- [Plant](terms/Plant.html)
- [accessLevel](terms/accessLevel.html)
- [ritualContext](terms/ritualContext.html)
